2024 Spring 網際內容管理與協同產品設計實習課程教學導引.

協同產品設計實習第十二週實習操作

  1. 鋼球平衡台零組件繪製與組立
  2. 使用 NX 1872 繪製鋼球平衡台所需的零組件。
  3. 參考影片及程式指南，組員分工完成零組件的繪製及組立。
  4. 每組至少提交一份零件繪製的 NXOpen Python 程式和一份零件組立的 NXOpen Python 程式。

  5. 在個人課程網誌中紀錄實習任務一的執行心得。

  6. 倉儲設定與整合

  7. 將各分組倉儲設定為評分網站子模組。
  8. 根據指示將分組倉儲設為評分倉儲的子模組。
  9. 整合全班各組項下各學員的網誌文章 (.html 格式) 與執行專案報告 (.pdf 格式)。

  10. 在各組的分組網誌與分組報告中說明如何將分組整合流程擴大到全班協同。

  11. 跨網路操控足球機器人競技

  12. 下載指定的球場模擬倉儲。
  13. 使用 CoppeliaSim 進行球場模擬。
  14. 使用遠端 ZMQ RemoteAPI Python 程式操控足球機器人進行球賽競技。
  15. 分別擔任球員角色，在各自電腦上使用鍵盤控制機器人進行競技。

  16. 每組確保能夠跨 IPv4 與 IPv6 網路進行操控，並設定防火牆開放對應的通訊埠。

  17. Web-based 協同輔助設計程式

  18. 使用 Flask 從教務主機取得班級課表的網際程式。
  19. 包括從網頁表單或 URL 網路連結取得輸入變數。
  20. 透過伺服器上的電腦執行後續存檔或運算。
  21. 應用於 Gogs 或 NXopen Python 等相關領域。

以上是本周的實習任務大綱，請確保完成並在指定時間內提交相關文件及程式碼。
