實施細節與步驟

一、先前影片整理

  1. 字幕添加與上傳
  2. 使用字幕軟件（如 Aegisub）添加字幕

  3. 上傳到 YouTube 或其他可嵌入的平台

  4. 逐字稿整理

  5. 從字幕文件中提取逐字稿

  6. 生成 .txt 文件並上傳

  7. 摘要撰寫

  8. 撰寫每個教學影片的摘要
  9. 將摘要放置在影片嵌入下方

二、分組網誌統整

  1. 修改 pelicanconf.py
  2. 設定 PATH 與 OUTPUT_PATH 變數

  3. 動態複製子模組 .md 文件到 combined_directory

  4. 轉檔操作

  5. 執行 pelican -s local_publishconf.py

  6. 命名與分類

  7. 網誌文件前綴為 "學號_"
  8. Category 使用學號標示

三、NX 繪圖

  1. 啟動 NX

  2. 使用 start_USB_nx1872.bat 啟動 USB 中的 NX1872

  3. 設置 temp 目錄

  4. 在 NX1872 目錄中新增 temp 目錄

四、ODOO PLM 論文翻譯

  1. 分工翻譯

  2. 各組員在個人倉儲中完成分工翻譯

  3. LaTeX 編譯

  4. 使用 miktex-portable 編譯 LaTeX 報告

  5. 生成 PDF

  6. 將最終報告生成 2a-midagx_report.pdf

五、Replit SSH 設定

  1. SSH 設置
  2. 在 Replit 中配置 SSH 管理分組倉儲

六、Sourcetree 使用

  1. 提交與回復
  2. 查看歷史提交
  3. 回復到任一版本
